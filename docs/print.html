Game Engine Docs

Introduction

The following chapters contain documentation on the game engine project I am working on. It also serves some other purposes, such as a reference point for me to see how I have improved and changed since this was made, as well as a sort of instructions book for myself and and others who find it useful.

The Core section goes over lessons learned from making the lower level core library (before working on the rest of the engine), extending functionality of the language to make things easier (for example, adding portable coroutines).

The API section is an API reference for the whole engine (including some core components), mostly serving as documentation, with some implementation details where necessary.

Core

I have chosen to move all of the utilities that wrap around the C++ and SFML interface into Core, keeping the foundations separate from the rest of the engine.

The following chapters are much more in-depth than the API reference, and explain the concept behind each method, why it is useful for the engine, some implementation details and other sources if required.

Some parts of Core are already documented in the API reference (e.g. Durations), and some future parts will be too, as they are much simpler, and don't require learning new concepts to create.

Time::Duration::Duration

Constructs a new duration, converting the time from the internal class to nanoseconds. time_base is the base class of Duration's member classes, decresing the required constructors to one.
